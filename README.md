# Personal Portfolio Website

A clean, modern, and responsive portfolio website built with HTML, CSS, and JavaScript.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Animated sections with fade-in effects
- Clean and professional layout
- Easy to customize
- Ready for interactive enhancements

## Sections

1. **Hero/Home** - Introduction with call-to-action buttons
2. **About** - Personal introduction and background
3. **Projects** - Showcase your work with descriptions and links
4. **Skills** - Display your technical skills and expertise
5. **Contact** - Links to your social media and email

## Customization

### Personal Information

Edit `index.html` to update:
- Your name and title in the hero section
- About me text
- Project details (title, description, technologies, links)
- Skills and technologies
- Contact information and social media links

### Colors and Styling

Edit `style.css` to customize:
- Color scheme (see CSS variables in `:root`)
- Fonts and typography
- Spacing and layout
- Animations and transitions

### Adding Interactivity

The `script.js` file is set up for future enhancements. You can add:
- Particle effects
- Canvas animations
- Interactive visualizations
- Dynamic content loading
- And more!

## Deployment

This site is ready for GitHub Pages:

1. Push your changes to the `main` branch
2. Go to your repository settings on GitHub
3. Navigate to "Pages" section
4. Select `main` branch as the source
5. Your site will be live at `https://hseoyeong.github.io`

## Local Development

Simply open `index.html` in your browser to view locally, or use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js
npx http-server
```

Then visit `http://localhost:8000` in your browser.
